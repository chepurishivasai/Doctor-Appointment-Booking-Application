# Doctor Appointment Booking Application

## Description
The Doctor Appointment Booking Application allows patients to book, manage, and confirm appointments with doctors easily. It provides a simple and user-friendly interface where users can select doctors, schedule appointments, simulate payment, and download confirmation details as a PDF.

---

## Features
- Doctor list with specialization and details  
- Appointment booking form with validation  
- LocalStorage integration for saving appointments  
- Simulated payment screen  
- Email confirmation simulation  
- Download Appointment PDF (jsPDF integration)  
- Fully responsive and modern UI  
- Separate pages for Home, Book Appointment, Appointments, and Contact  

---

## Technologies Used
- HTML5  
- CSS3  
- JavaScript (ES6)  
- jsPDF (for PDF generation)  
- LocalStorage (for temporary data storage)  

---

## How to Run
1. Download or clone the repository  
2. Open `index.html` in your browser  

---

## Project Structure
- index.html → Home page  
- book.html → Appointment booking page  
- appointments.html → View appointments  
- contact.html → Contact page  
- style.css → Styling  
- script.js → Functionality
